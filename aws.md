# Database 
## RDS: Relational Database Service 
Create and config relational db running on EC2.
* software upgrades
* monitoring
* periodic backups

Features:
* Backups
* Multi-AZ deployment: 
  create a replica in another AZ in the same Region
* Create a Read replica of the db (for non production uses, eventually consistent)

Some Data engines:
* MySQL
* PostgreSQL
* Microsoft SQL Server
* MariaDB
* Oracke
* Amazon Aurora
